Podcast Studio

자료를 올리면 2인 대화 스크립트와 더빙 오디오를 만듭니다.

자료 입력

텍스트, 마크다운, CSV, JSON, HTML 파일을 지원합니다. PDF/DOCX는 텍스트로 복사해서 붙여넣으면 가장 안정적입니다.

API 설정 · 오디오 생성

스크립트 생성은 OpenAI 또는 Claude(Anthropic) API로 할 수 있습니다. 오디오(TTS) 저장은 OpenAI 전용이므로 WAV를 만들려면 OpenAI 키가 필요합니다. 키는 저장하지 않고 현재 화면에서만 사용합니다.

브라우저 음성

설치된 브라우저/운영체제 음성 목록입니다. API 키 없이 미리듣기할 때 사용됩니다.
